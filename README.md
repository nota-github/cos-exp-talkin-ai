# Talkin AI

한국어 사용자에게 최적화된 데스크탑 AI 에이전트 앱을 개발하고 싶다.

이 서비스는 사용자가 한국어로 자연스럽게 프롬프트를 입력하면, 내부적으로 한국어 프롬프트를 AI 모델이 더 저렴하고 효율적으로 처리할 수 있는 영어 기반 토큰 흐름으로 변환한 뒤 GPT, Claude, Gemini 같은 모델에 전달한다. 모델 응답은 다시 자연스러운 한국어로 변환해 사용자에게 보여준다.

핵심은 사용자가 영어를 직접 쓰게 만드는 것이 아니라, 한국어 사용 경험은 그대로 유지하면서 내부 모델 호출 비용과 토큰 사용량을 줄이는 것이다.

비용 최적화 엔진은 로컬 LLM 기반으로 동작해야 한다.

사용자가 한국어 프롬프트를 입력하면, 데스크탑 앱 내부의 로컬 LLM이 먼저 한국어 의도와 맥락을 보존한 채 클라우드 AI 모델이 더 적은 토큰으로 처리할 수 있는 영어 기반 프롬프트로 변환한다. 이후 GPT, Claude, Gemini 같은 클라우드 모델에는 최적화된 영어 프롬프트만 전달된다.

클라우드 모델의 영어 응답은 다시 로컬 LLM이 자연스러운 한국어로 변환해 사용자에게 보여준다.

이 구조의 목적은 비용 절감을 위해 번역 자체에 또 다른 유료 API 비용을 쓰지 않는 것이다. 즉, 한국어-영어 변환과 영어-한국어 복원은 로컬에서 처리하고, 고비용 클라우드 모델 호출은 실제 추론이 필요한 핵심 작업에만 사용한다.
초기 제품은 뤼튼, ChatGPT, Claude 같은 채팅형 데스크탑 앱 경험을 제공하되, 내부에는 Translation MCP 기반의 한국어 비용 최적화 엔진을 탑재한다. 이후 여러 에이전트 작업을 동시에 관리할 수 있는 멀티채팅(split view) 및  Kanban UI로 확장할 수 있다.
