Then extend `apps/desktop/tests/usage-dashboard.test.ts:122-338` with a seeded mixed-provider/model/pricing-version case so `story-4.3:VAL-1` proves the dashboard never computes aggregate savings from mixed bases, and refresh `_bmad-output/cos-agent/implementation/run_932c568d-64e0-484b-9621-62b4891d907e.md` so its claimed acceptance evidence matches the shipped behavior.
